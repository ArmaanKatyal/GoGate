Features that it will have:

-   [] URL Rewriting
-   [] Load Balancing
-   [] Rate Limiting / Throttling / Quotas
-   [] Caching
-   [] SSL Termination
-   [] Orchestration/Aggregation
-   [] Protocol Translation
-   [] IP Whitelisting
-   [] Retry Policy, Circuit Breaker
-   [] Logging, Tracing
